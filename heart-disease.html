❤️ Heart Disease Prediction

A machine learning pipeline to predict heart disease risk using clinical features. The project includes data cleaning, feature engineering, model training (Logistic Regression / Random Forest), evaluation and a Streamlit UI to input patient parameters and get an instant prediction.

Technologies

  • Python · Pandas · NumPy
  • Scikit-learn · Matplotlib
  • Streamlit for deployment

Highlights

  • Trained on publicly available heart disease datasets
  • Exploratory data analysis, feature selection & model evaluation
  • Clean Streamlit frontend for quick input & inference
